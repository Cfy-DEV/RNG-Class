# RNG Class
A roblox lighweight module that lets you store "Items" with certain percentages in them to later on roll, just like a gacha of some sort.

## Getting Started
To use it in your projects, i'd recommend adding it to your project's wally.toml file :

```bash
rng-class = "cfy-dev/rng-class@1.0.0"
```
